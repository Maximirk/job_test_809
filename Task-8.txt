Задача: Опишите как бы вы реализовали взаимодействие системы Библиотека с системой Учета,
в которой есть операции Купить книгу, Дать книгу в пользование, Просмотреть каталог книг,
но при этом информация о книгах хранится в системе Библиотека

ОТВЕТ:
Для системы учета книг, можно создать свою таблицу в БД, где будут различные данные. Например цена,
можно ли ее продавать или только давать, количество, можно ли отображать в каталоге и т.д. Эта информация будет связана
с библиотечной таблицей о книгах с помощью id_book. Можно реализовать свой интерфейс для учетной системы, где можно
создать все эти данные для уже имеющихся книг в библиотеке.

Для покупок книг создать две таблицы в БД, одна для информации по заказам, другая для информации о заказанных
товарах с привязкой к заказам.

Тоже самое для сдачи книг в аренду.

При просмотре каталога, получать данные из двух таблиц, из библиотеки и системы учета и отображать с учетом обоих
массивов данных.
